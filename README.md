# Organisation d'astreinte

## Installation

- Cloner le repos:
```git clone https://github.com/LauraPMS/smart-calendar.git```

- Installer Cargo et rustup :
> #Windows
> Se rendre sur le site offciel rustup.rs et télécharger le rustup-init.exe
> Suivre les instructions
> Recharger le terminal

- Lancer l'application avec Cargo :
```cargo run```

- Cela va créer un serveur a l'adresse : 
```http://127.0.0.1:3000/```
